##APLICACION DEL CLIMA 
Este es un repositorio para una aplicacion que introducimos por la consola
nos muestra la temperatura y modificacando el repositorio podemos mostrar ademas 
las coordenadas geograficas de la ciudad que introdusco por consola

PARA UTILIZAR NO OLVIDAR 
 
 ```npm install```

 ##Ejemplo

 ````node app -d "Monteria Colombia" ````
